Privacy Policy & Opting Out

Introduction

The privacy policy outlines exactly what data is collected, and how you can view, manage, or delete it. If you're in a server running this bot, the stats pages are the only place your data will ever be used. Your data will never be shared with any third-party, ever.


How is my data kept safe?

To see statistics for a server, your account must be in that server. This information is verified when you login using your Discord account. Unfortunately, this means that if you leave a server, its statistics will be hidden from you.
Because Discord users are prone to leaving and rejoining servers, your data will not be deleted if you leave a server. If you want to delete it, you may do so via the dashboard for that server.

The data itself is securely hosted in an Amazon RDS database.

What data is collected?

Here's a list of all data retained by the bot. For servers without the 'server history' feature enabled, this only applies to messages sent after the bot joined.

Data collected per server

  • Server Name
  • Server ID
  • Server Icon URL
  • Names of viewable text channels
  • IDs of viewavle text channels

Data collected per user

  • Tag (your username)
  • User ID
  • Avatar URL
  • Number of messages per viewable text channel
  • Average message length
  • Curse word frequency - curse words themselves are not collected

Data collected per user message

  • "Uncommon" words - words that do not appear in a standard dictionary
  • Users mentioned
  • Message date and hour
  • URLs used
  • Emojis used
  • Reactions


Opting Out - Hiding or Deleting Your Data

If you aren't comfortable with the bot displaying or retaining your data, you can manage the data associated with your account using a few simple commands.

Hiding your data

Using this command means that any mentions of your account in the server's stats will be hidden. However, your data will still anonymously count towards server-wide statistics. This action can be undone. All of your data will be retained, but not shown.

Deleting your data

Using this command deletes your data. This action cannot be undone! All data associated with your account will be permanently erased. The bot will subsequently ignore all activity associated with your account.

What if my account is no longer in the server?

The opt-out commands may still be used if you DM the bot.

What if I deleted my account?

Unfortunately, Discord does not alert bots when accounts are deleted.
