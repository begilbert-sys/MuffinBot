MuffinStats

Detailed Statistics for Discord Servers

MuffinStats offers a wide array of options for viewing statistics about your Discord server, right here on this website.

  • Personalized - Aside from generating statistics for each server, each user gets their very own statistics profile
  • Secure - Statistics are private, and you can select exactly which members of your server are allowed to view them
  • Privacy-Friendly - Each user has complete control over any data collected by the bot
  • Intuitive - Setting up the bot is simple and quick
