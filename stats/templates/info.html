Additional Info

Credits

MuffinStats is heavily influenced by mIRCStats, a program that served the same purpose for IRC chatrooms back in the day. As for development, MuffinStats is written mostly in Python, with indespensible help from the following third-party tools:

  • discord.py - a library for handling interaction with Discord's API
  • Django - a framework for database and website management

Both of these resources have beautifully-written documentation and made the development of this project much easier.

Aside from that, MuffinStats is created and maintained solely by @muffinlicious on Discord.

Bug Reporting & Suggestions

The project is still in beta, so expect things to occasionally go wrong. If you encounter any bugs, or want to suggest an idea, please DM me on Discord (@muffinlicious), or email me at muffinstats@gmail.com. Any help is greatly appreciated!
