Additional Info

Credits

MuffinStats was heavily influenced by mIRCStats, a program that served the same purpose for IRC chatrooms back in the day. As for development, MuffinStats was written mostly in Python, with indespensible help from the following third-party tools:

  • discord.py - a library for handling interaction with Discord's API
  • Django - a framework for database and website management

Both of these have beautifully-written documentation and made the development of this project much easier.

Aside from that, MuffinStats was created by and is maintained solely by @muffinlicious on Discord.

Bug Reporting

The project is still in beta, so expect things to occasionally go wrong. If you encounter any bugs, please DM me on Discord (@muffinlicious), join my server, or email me at muffinstats@gmail.com. Any help is greatly appreciated!
